# IPL-2008-2025-Dashboard
## IPL 2025 Data Analytics Project 📊
   This project analyzes IPL 2008-2025 points table and team performance data, showcasing end-to-end data analytics skills using SQL Server and Power BI. It demonstrates loading, cleaning IPL datasets, running SQL queries for insights, and building interactive dashboards—ideal for data analyst portfolios.
​

## 📋 Project Overview
   - Analyzes IPL 2025 team standings like Royal Challengers Bengaluru (RCB) leading with strong wins, alongside Punjab Kings, Gujarat Titans, and others from the provided screenshot. 
   - Focuses on key metrics: matches played, wins, losses, and points.
  
## 🛠️ Tech Stack
- SQL Server: For data querying and analysis.

- Power BI: For dashboard creation and visualizations.

- Python/Excel: Initial data loading and cleaning .
​
## 📥 Dataset
   - IPL 2025 points table data from official sources and screenshots, including teams like RCB (14 points from 10 matches), Punjab Kings, Sunrisers Hyderabad, and more. Covers matches played (P), wins (W), losses (L), and totals. Publicly available IPL data (2008-2025) used for completeness.
​

## 🚀 Getting Started
- Clone the repo and open in VS Code or SQL Server Management Studio.

- Load dataset into SQL Server via import wizard.

- Run cleaning scripts, execute queries, then connect to Power BI.

- Refresh dashboard for live insights—no installs needed beyond tools.


## 🔄 Workflow
Load Data 📥: Import CSV/Excel IPL data to SQL Server.

Clean Data 🧹: Handle nulls, standardize team names.

SQL Queries ⚙️: Write queries for top teams, win rates.

Power BI Dashboard 📈: Model data, create visuals like bar charts for points, tables for standings, slicers for teams.
