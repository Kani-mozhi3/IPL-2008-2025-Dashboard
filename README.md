# IPL-2008-2025-Dashboard
## IPL 2025 Data Analytics Project 📊
This project analyzes IPL 2025 points table and team performance data, showcasing end-to-end data analytics skills using SQL Server and Power BI. It demonstrates loading, cleaning IPL datasets, running SQL queries for insights, and building interactive dashboards—ideal for data analyst portfolios.
​

## 📋 Project Overview
Analyzes IPL 2025 team standings like Royal Challengers Bengaluru (RCB) leading with strong wins, alongside Punjab Kings, Gujarat Titans, and others from the provided screenshot. Focuses on key metrics: matches played, wins, losses, and points.
